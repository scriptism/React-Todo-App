# Modern Todo App  
A sleek, responsive todo application built with **React** and **Tailwind CSS**.

## Features  

✔️ Add, edit, and delete tasks  
✔️ Toggle task completion  
✔️ Persistent storage (localStorage)  
✔️ Smooth animations  
✔️ Mobile-friendly design
